Quick notes

Use GitHub Pages, not the normal github.com file preview URL.

In Docebo, create an iframe additional field for a course, paste the GitHub Pages URL, set a height like 700, then test the course properties page.

If a message arrives, the debug panel below will update. If not, your iframe may still be loading correctly; it just means no message was sent in that moment.
